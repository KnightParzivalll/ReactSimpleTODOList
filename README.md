# TODO List

A simple React todo list built with **Mantine UI** that supports adding, editing, deleting, and reordering tasks. Includes **dark/light theme** and persists data in **localStorage**.

---

## Screenshot

![TODO List Screenshot](./screenshots/screenshot.png)

---

## Features

- Add new tasks with title and description
- Edit existing tasks
- Delete tasks
- Move tasks up and down
- Dark/light theme toggle
- Local storage persistence

---

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/KnightParzivalll/ReactSimpleTODOList.git
cd ReactSimpleTODOList
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```
